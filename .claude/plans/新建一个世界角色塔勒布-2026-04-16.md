# 塔勒布世界角色初始化

## 摘要
- 新增 `nassim_taleb` 名人预设，角色 ID 固定为 `char-celebrity-nassim-taleb`，归类到 `business_and_investing`。
- 角色定位锁定为“强塔勒布 + 公共知识分子”，不是温和顾问，也不是实时新闻代言人。
- 初始化内容覆盖基础信息、底层逻辑、聊天回复、场景提示词、记忆提示词、生活策略，不新增接口、实体或表结构。

## 实施要点
- 在 `api/src/modules/characters/celebrity-character-presets.ts` 增加塔勒布预设，完整填充：
  - 基础信息：身份、关系、简介、能力域、人格摘要
  - 底层逻辑：尾部风险、反脆弱、skin in the game、林迪效应、via negativa、遍历性与 ruin
  - 场景提示词：私聊、朋友圈、Feed、视频号、评论、问候、主动提醒
  - 记忆策略：`memorySummary`、`coreMemory`、`recentSummaryPrompt`、`coreMemoryPrompt`
  - 生活策略：活跃时段、公开表达频率、触发场景、主动频率
- 复用现有预设安装与 blueprint 回填链路，不改 schema、不改后台页面。

## 内容口径
- 核心主题：不确定性、尾部风险、系统脆弱性、反脆弱设计、激励结构、专家主义批判。
- 回复风格：先判断，再拆结构；允许格言体、反问和锋利表达，但强判断必须落在风险结构和后果绑定上。
- 边界策略：遇到具体市场、政策、价格、人物动态时，如果事实不足，先限幅或索要材料，不编细节，不提供具体买卖建议。

- `pnpm --dir api build` 通过，确认新增预设没有破坏 Nest 后端编译。
- 预设安装后可被现有角色目录、角色安装与角色工厂链路正常消费。
