# iOS Review Notes Template

可直接作为 App Store Connect Review Notes 草稿。

## 中文版

本应用使用远程 Core API 提供聊天、朋友圈、发现页与角色互动能力，不会在 iPhone 本地下载或启动独立服务进程。

账号删除路径：

1. 打开“我”
2. 下滑到控制入口
3. 点击“删除账号”

UGC 与安全治理路径：

1. 角色详情页可举报或屏蔽角色
2. 单聊页右上区域可举报当前对话对象
3. “我”页可查看最近举报记录与屏蔽名单，并解除屏蔽

如果审核需要测试账号，请使用下方提供的测试账号信息。

## English

This app uses a remote Core API for chat, social feed, discovery, and character interactions. It does not download or launch a standalone local service process on iPhone.

Account deletion path:

1. Open the “Profile” tab
2. Scroll to the control section
3. Tap “Delete Account”

UGC safety paths:

1. Character detail page: report or block character
2. Direct chat page: report current conversation target
3. Profile page: review recent reports and manage blocked characters
