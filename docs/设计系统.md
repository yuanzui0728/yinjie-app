## 设计原则

隐界的 UI 设计遵循微信的视觉语言，让用户打开第一秒就知道怎么用。

- **熟悉感优先**：颜色、间距、导航结构与微信高度一致
- **克制**：不过度设计，信息密度适中
- **温度感**：emoji 头像、暖色调互动元素，传递"真实社交"的感觉

## 圆角规范

| 场景 | 圆角值 |
|------|--------|
| 头像 | `size / 5`（自适应） |
| 消息气泡 | 18，发送方底角 4 |
| 卡片 | 10-12 |
| 标签/Badge | 4-20（根据形状） |
| 按钮 | 8-10 |

## 导航结构

```
App
└── Stack Navigator
    ├── (tabs)                    # 底部 Tab 导航
    │   ├── index (消息)          # 聊天列表
    │   ├── moments (朋友圈)      # 朋友圈 Feed
    │   ├── contacts (通讯录)     # 角色列表
    │   └── profile (我)          # 个人页
    ├── chat/[id]                 # 聊天页（单聊+群聊）
    ├── character/[id]            # 角色详情
    ├── character/create          # 创建角色
    └── import/index              # 导入聊天记录
```
