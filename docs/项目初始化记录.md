# 项目初始化记录

> 完成时间：2026年3月11日

## 做了什么

初始化了隐界 App 的前端项目，基于 Expo + React Native + TypeScript。

## 技术选型

- **Expo SDK 55** + React Native 0.83
- **expo-router** — 文件路由（类 Next.js）
- **zustand** — 全局状态管理
- **react-native-safe-area-context** — 安全区域适配

## 关键决策

1. **用 expo-router 而非 react-navigation**：文件路由更直观，减少配置代码
2. **Mock 数据驱动**：所有页面先用 Mock 数据跑通，后续替换为真实 API
3. **zustand 而非 Redux**：轻量，无样板代码，够用
4. **TypeScript 零错误**：`npx tsc --noEmit` 通过

## 启动方式

```bash
cd mobile
npm start
# 扫码用 Expo Go 预览，或 npm run android / npm run ios
```
