## 页面列表

### Tab 页面

#### 聊天列表（app/(tabs)/index.tsx）
- 顶部导航栏：标题"隐界" + 加号按钮
- 搜索框（UI 层，暂无功能）
- 会话列表：按 updatedAt 降序排列
- 未读数红点显示在 Tab 图标上

#### 朋友圈（app/(tabs)/moments.tsx）
- 顶部封面图（深色背景 + 品牌文案）
- 用户头像悬浮在封面右下角
- 动态列表：按 postedAt 降序排列
- 相机按钮（UI 层）

#### 通讯录（app/(tabs)/contacts.tsx）
- 分组展示：数字亲人 / 专家朋友团 / 朋友
- 每个角色显示：头像 + 姓名 + 关系标签 + 专长 Badge + 在线状态
- 右上角加号 → 跳转创建角色页

#### 个人页（app/(tabs)/profile.tsx）
- 用户信息卡片
- 菜单列表（收藏/朋友圈/支付/设置）
- 导入聊天记录入口 → 跳转 /import

### 功能页面

#### 聊天页面（app/chat/[id].tsx）
- 支持单聊和群聊（通过 conversation.type 区分）
- 消息列表：MessageBubble 组件渲染
- 发送消息后 1.5 秒显示 TypingIndicator，然后 Mock AI 回复
- 键盘弹起时 KeyboardAvoidingView 适配
- 点击右上角头像 → 跳转角色详情页

#### 角色详情页（app/character/[id].tsx）
- 头像 + 姓名 + 关系标签 + 专长 Badge
- 简介和性格描述
- "发消息"按钮

#### 创建角色（app/character/create.tsx）
- 头像选择（12 个 emoji 选项）
- 姓名输入
- 关系标签输入
- 关系类型选择（家人/朋友/专家/自定义）
- 专长领域多选
- 性格描述输入
- 导入聊天记录入口

#### 导入聊天记录（app/import/index.tsx）
- 4 步流程：intro → select → processing → done
- processing 步骤有进度条动画（Mock，5 步，约 5 秒）
- 完成后跳转到妈妈的聊天页

## 组件列表

### character/Avatar.tsx
```tsx
<Avatar emoji="👩" size={44} showOnline isOnline={true} onPress={() => {}} />
```
- `emoji`：显示的 emoji
- `size`：尺寸（默认 44）
- `showOnline`：是否显示在线状态点
- `isOnline`：在线状态（绿点/灰点）

### character/ExpertBadge.tsx
```tsx
<ExpertBadge domain="law" />
```
- 支持 8 种专长：law / medicine / finance / tech / psychology / education / management / general
- 每种专长有独立颜色

### chat/MessageBubble.tsx
```tsx
<MessageBubble message={message} />
```
- `senderType === 'user'`：右侧绿色气泡
- `senderType === 'character'`：左侧白色气泡 + 发送者名字
- `type === 'system'`：居中灰色系统消息（如"XX 加入了群聊"）

### chat/TypingIndicator.tsx
```tsx
<TypingIndicator />
```
- 三个跳动的点，模拟 AI 正在输入

### chat/ChatListItem.tsx
```tsx
<ChatListItem conversation={conv} onPress={() => {}} />
```
- 头像 + 名字 + 最后一条消息预览 + 时间 + 未读数

### moments/MomentCard.tsx
```tsx
<MomentCard moment={moment} />
```
- 头像 + 作者名 + 时间 + 位置
- 动态文字内容
- 点赞/评论交互（长按"···"弹出操作菜单）
- 互动区域：AI 之间的点赞和评论

## 状态管理

### chatStore.ts
- `conversations`：会话列表
- `messages`：消息记录（按 conversationId 分组）
- `sendMessage(id, text)`：发送消息 + 触发 Mock AI 回复
- `markAsRead(id)`：标记已读

### momentsStore.ts
- `moments`：朋友圈动态列表
- `toggleLike(id)`：切换点赞状态
- `addComment(id, text)`：添加评论

### characterStore.ts
- `characters`：角色列表
- `getById(id)`：按 ID 查找角色
