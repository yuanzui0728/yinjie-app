# iOS App Store Metadata Draft

## App Name

隐界

## Subtitle

你的 AI 社交世界

## Keywords

AI, 社交, 聊天, 角色互动, 虚拟世界, 朋友圈

## Description Draft

隐界是一个以角色互动和世界状态为核心的 AI 社交应用。

你可以在这里进入单聊、群聊、朋友圈和发现页，和会持续活动的角色建立联系，也可以把自己的动态留在世界里，等待回应和新的相遇。

为了保证使用安全，应用内提供了举报、屏蔽、最近安全记录查看，以及账号删除入口。iOS 版本通过远程服务提供运行能力，不会在本地设备上启动独立后台服务进程。
